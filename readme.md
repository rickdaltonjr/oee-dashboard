# 🏭 Dashboard OEE - Indústria 4.0

Sistema de monitoramento de Eficiência Global de Equipamentos (OEE) para indústria de laticínios, desenvolvido com Python, SQLite e Streamlit.

![OEE Dashboard](https://img.shields.io/badge/OEE-Real--time-blue)
![Python](https://img.shields.io/badge/Python-3.9+-green)
![Streamlit](https://img.shields.io/badge/Streamlit-1.28+-red)

## 📊 Funcionalidades

- **KPIs em tempo real**: OEE, Disponibilidade, Performance, Qualidade
- **Filtros dinâmicos**: Data, Linha de produção, Turno
- **Gráficos interativos**: Evolução temporal, Comparativo por linha
- **Análise de Pareto**: Top causas de parada (80/20)
- **Detecção automática de schema**: Adapta-se a diferentes estruturas de banco
